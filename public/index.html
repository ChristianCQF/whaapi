WhatsApp Service - Render API

Estado: Cargando...

Endpoints API:

GET  /              - Estado del servicio
GET  /status        - Información detallada
GET  /qr            - Obtener código QR
POST /send-message  - Enviar mensaje WhatsApp

Ejemplo de envío:

curl -X POST https://tu-app.onrender.com/send-message \
  -H "Content-Type: application/json" \
  -d '{
    "phoneNumber": "54911XXXXXXXX",
    "message": "Hola desde Render!"
  }'
